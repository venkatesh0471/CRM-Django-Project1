Employee CRM Management System (Django)

A complete Employee CRM (Customer Relationship Management) web application built using Django, Python, HTML, CSS, and JavaScript.
This project allows you to Create, Read, Update, and Delete (CRUD) employee details with image upload support and a clean UI.

 Features

 Add Employee (name, email, contact, subject/department, image)

 Update Employee Details

Delete Employee Records

 View All Employees in a dynamic table

 Image Upload using Django Media

 Admin Panel Integration (Django Admin)

 Responsive UI using HTML + CSS + JS

 Uses MVT architecture of Django

Tech Stack
Area	Technology
Backend	Django, Python
Frontend	HTML, CSS, JavaScript
Database	SQLite / MySQL
Architecture	MVT (Model-View-Template)
Tools Used	VS Code, Git, GitHub
